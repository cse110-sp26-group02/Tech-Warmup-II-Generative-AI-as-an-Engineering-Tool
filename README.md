# Tech-Warmup-II-Generative-AI-as-an-Engineering-Tool

**Model**: `gemini-3.1-pro-preview`

**Harness**: `Gemini CLI`

## Setup

To install necessary development dependencies (ESLint for checking JS code), run the following command from the root of this repository:

```bash
npm install
```

## Linting

To check the JavaScript files for code quality and style issues, run the following command:

```bash
npm run lint
```
