# Tech-Warmup-II-Generative-AI-as-an-Engineering-Tool

**Model**: `gemini-3.1-pro-preview`

**Harness**: `Gemini CLI`


## Model Selection Rationale

*Why we chose gemini-3.1-pro-preview*

For this project, we utilized **Gemini 3.1 Pro Preview**. We chose this model because it is the latest and most capable model available. We wanted to leverage the most advanced/up-tp-date AI technology possible to understand the full potential of generative AI in a software engineering context. Using this model ensured we were working with the best available tool for code generation, problem-solving, and following complex instructions.
